## Overview

ePay Wallet is a secure, role-based digital wallet application inspired by bKash and Nagad. Users can perform wallet operations including deposit, withdrawal, sending money, and cash in/out, with real-time transaction tracking.

## Key Features

- **JWT-based authentication** with role-based dashboards (Admin, User, Agent)
- **Wallet operations**: Deposit, Withdraw, Send Money, Cash In/Out
- **Transaction history** with filters, charts, and pagination
- **Admin controls**: Manage users, approve agents, block/unblock wallets
- **OTP verification** for secure operations
- **Responsive design** with dark/light mode toggle
- **Guided tours** and toast notifications
- **Atomic balance updates** for secure transactions

## Case Study

ePay Wallet is a full-stack digital wallet system designed to simulate real-world financial operations. The frontend is built with React and TypeScript, using Redux Toolkit and RTK Query for scalable state management and efficient API communication.

It features JWT-based authentication, role-based dashboards for Admin, User, and Agent, responsive design, dark/light mode toggle, and guided tours to improve user experience.

The backend is developed using Node.js, Express.js, and MongoDB (Mongoose) with modular, layered architecture. Key features include wallet auto-creation with initial balance, secure transaction operations, transaction history with filters and pagination, and OTP verification via email.

Admins can manage users, approve agents, and block/unblock wallets, with atomic balance updates ensuring financial security. Additional enhancements include file uploads using Multer and Cloudinary, toast notifications, and charts for transaction visualization.
