## Overview

Gadget Heaven is a frontend e-commerce application where users can browse gadgets, view detailed product information, add items to the cart or wishlist, and manage their selections with persistent state using local storage.

## Key Features

- **Active route system** with React Router for smooth navigation
- **Add to cart functionality** with persistence via Local Storage
- **View detailed gadget information** for each product
- **Remove items from cart** with instant UI feedback
- **Add and manage wishlist items** with duplicate prevention

## Case Study

Gadget Heaven is a React-based e-commerce web application focused on building a realistic shopping experience using modern frontend architecture. The primary goal was to strengthen core React fundamentals such as component-based design, hooks, routing, and global state management.

The application uses React Router to implement an active route system, allowing smooth navigation across different sections. Users can view detailed information about each gadget, add products to the cart or wishlist, and remove items when needed.

Application state is managed using React Context API, while Local Storage is used to persist cart and wishlist data across browser sessions. Tailwind CSS and DaisyUI create a clean, responsive user interface.

This project demonstrates strong understanding of React fundamentals, client-side state management, and user-focused feature design.
