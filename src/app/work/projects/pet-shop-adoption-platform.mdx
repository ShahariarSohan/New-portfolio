## Overview

Pet Shop is an interactive pet adoption web application where users can explore different pet categories, view detailed information about each pet, like their favorite pets, and sort listings by price using modern JavaScript ES6 features.

## Key Features

- **Browse pets by category** with dynamic filtering
- **View detailed information** for each individual pet
- **Display liked pets** dynamically on interaction
- **Sort pets by price** for easy comparison
- **Like and favorite pets** with instant UI updates
- **Responsive and interactive UI** without any framework

## Case Study

Pet Shop is a frontend-focused web application designed to simulate a pet adoption experience. The goal was to practice core JavaScript concepts while building an interactive and user-friendly interface.

Users can browse pets by category, view detailed information about individual pets, and mark pets as favorites using a like feature. Liked pets are instantly displayed on the screen, enhancing user engagement. The application also allows sorting pets by price.

The project makes use of modern ES6 features such as arrow functions, async functions, array methods like forEach and sort, and controlled timing with setTimeout. This helped ensure cleaner, more readable, and maintainable code.

Overall, this project demonstrates strong fundamentals in JavaScript, DOM manipulation, and frontend logic, while focusing on usability and interactive behavior.
