## Overview

Select Your Dream XI is a cricket team building application where users can create their own fantasy team by selecting players within a limited coin balance. The project focuses on interactive state management, validation logic, and smooth user feedback.

## Key Features

- **Build a custom Dream XI** cricket team with real player data
- **Add and manage virtual coin balance** as in-game currency
- **Select players with real-time validation** and budget checks
- **Prevent duplicate player selection** with client-side logic
- **View selected players** in a dedicated section
- **Dynamic and responsive** user interface

## Case Study

Select Your Dream XI is a frontend web application inspired by fantasy cricket platforms. The main objective was to practice React state management, user interaction handling, and validation logic in a real-world scenario.

Users are given a virtual coin balance that they can increase and spend to select players for their team. The application ensures that the same player cannot be selected more than once, enforcing proper business rules on the client side. Selected players can be viewed separately, and users can dynamically add more players.

The project is built using React for component-based architecture, Tailwind CSS and DaisyUI for clean and responsive styling, and React Toast for instant user feedback. This project demonstrates strong understanding of component state, conditional rendering, and user-centric UI behavior.
