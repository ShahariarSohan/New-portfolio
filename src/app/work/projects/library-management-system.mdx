## Overview

A full stack library management system where users can manage books, borrow available titles, and view borrowing summaries. Built with React and TypeScript on the frontend and a Node.js, Express, and MongoDB backend with strong validation and RESTful APIs.

## Key Features

- **Create, update, and delete books** with full CRUD support
- **View detailed book information** for each entry
- **Borrow books** and track availability in real-time
- **View borrowed book summary** across all users
- **Filter books by genre** for easy discovery
- **Client-side and server-side validation** with Zod

## Case Study

The Library Management System is a full stack web application designed to simulate a real-world library workflow, where users can manage book collections and borrowing activity efficiently.

The frontend is built using React and TypeScript, with Redux Toolkit Query (RTK Query) handling server state and API communication. The UI follows a clean and minimal design using Tailwind CSS and Shadcn component library.

The backend is developed using Node.js, Express, and TypeScript, with MongoDB and Mongoose for data persistence. Zod validation is implemented to ensure data integrity and prevent invalid input. The server exposes RESTful APIs to fetch all books, filter books by genre, retrieve books by ID, and manage borrowing records.

This project highlights strong understanding of full stack development, REST API design, data validation, and scalable state management.
