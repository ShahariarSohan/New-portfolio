## Overview

HDRT (Hidden Dependency Risk Tracker) is a full-stack, enterprise-grade risk tracking system designed to visualize and manage hidden operational dependencies in organizations. The platform provides role-based dashboards, real-time analytics, and comprehensive risk scoring for employees, teams, and critical systems.

## Key Features

- **Role-based dashboards** for Admin, Manager, and Employee roles
- **Secure authentication** with JWT and HttpOnly cookies
- **Server Components + Server Actions** for secure, real-time data fetching
- **Interactive risk visualization** with badges, progress bars, and score mapping
- **Centralized API service layer** for consistent and type-safe communication
- **Pagination, filtering, and searching** across all modules
- **Soft-delete strategy** for all entities (tasks, employees, teams, systems)
- **Modular, scalable, enterprise-ready** frontend and backend architecture
- **Task, team, system, and employee management**
- **Real-time operational insight dashboards**

## Case Study

HDRT is a full-stack risk analysis tool built for enterprise environments to proactively identify operational risks caused by excessive reliance on specific employees, teams, or systems.

The frontend is developed with Next.js 15 using React Server Components (RSC), Server Actions, TypeScript, Tailwind CSS, and Shadcn UI for a modular, scalable architecture. It delivers role-based dashboards for Admin, Manager, and Employee roles, including interactive risk visualizations, paginated tables, filtering, soft-delete indicators, and centralized API service layers.

The backend is built with Node.js, Express.js, TypeScript, PostgreSQL, and Prisma ORM. It features JWT-based authentication with role-based access control, Zod validation for secure input handling, soft-delete strategies to preserve historical data, and enterprise-level APIs for tasks, teams, systems, employees, and risk analytics.

Risk calculation logic includes weighted scoring for tasks, employees, teams, and systems, providing actionable operational insights.

This project demonstrates advanced full-stack capabilities, enterprise-grade architecture, secure and scalable backend design, and a strong focus on operational decision support through analytics dashboards.
